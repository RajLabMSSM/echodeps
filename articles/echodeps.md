## Gather package metadata

### echoverse

`package_metadata` pulls metadata from the DESCRIPTION files from a
package (or list of packages) and organises them into a convenient
table.

Metadata for the [*echoverse*](https://github.com/topics/echoverse)
suite of R packages will be retrieved by default. While *echoverse*
itself is not the name of an R package, `package_metadata` will
automatically recognise this special command and retrieve all packages
that are part of this suite.

``` r

meta_echoverse <- echodeps::package_metadata(pkgs = "echoverse")
```

## Create a dependency graph

`dep_graph` builds upon the above functions and creates a dependency
graph for all the dependencies of a given R package (or a selected
subset of its dependencies).

Metadata for each package is automatically extracted and shown as
tooltips that appear when you hover over the respective node.

All of the plots, graphs, metadata, and saved file paths generated by
`dep_graph` are returned to the user as a named list in case they would
like to explore them further.

### echolocatoR

``` r

res_echoverse <- echodeps::dep_graph(pkg = "echolocatoR",
                                     ## Setting show_plot for vignette purposes
                                     show_plot = list(r=TRUE,
                                                      browser=FALSE))
```

### dplyr

``` r

res <- echodeps::dep_graph(pkg = "dplyr",
                           shape = "hexagon",
                           layout = function(graph, pkg){
                               visNetwork::visIgraphLayout(
                                   graph = graph,
                                   layout = "layout_nicely"
                               )
                           }
                           )
```
